# 📊 Projeto de Análise de Vendas

Este projeto foi desenvolvido com o objetivo de praticar conceitos de análise de dados utilizando SQL e Power BI.

## 🔧 Ferramentas utilizadas
- SQL
- Power BI
- CSV (dados simulados)

## 📁 Estrutura do projeto
- /dados → base de dados em CSV  
- /sql → consultas SQL  
- /dashboard → arquivo do Power BI  

## 📌 Análises realizadas
- Total de vendas  
- Vendas por categoria  
- Produtos mais vendidos  
- Ticket médio  

## 📈 Principais insights
- Produtos eletrônicos possuem maior faturamento total  
- Periféricos apresentam maior volume de vendas  
- Notebook é o produto com maior impacto financeiro  

## 🚀 Objetivo
Projeto desenvolvido para demonstrar habilidades iniciais em análise de dados, manipulação de dados e criação de dashboards.
